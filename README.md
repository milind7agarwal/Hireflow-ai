# 🚀 HireFlow AI - https://hireflow-ai-ashy.vercel.app

An AI-powered hiring assistant that helps streamline the recruitment process by analyzing resumes, matching candidates with job descriptions, and generating intelligent interview insights.

---

## 📌 Overview

HireFlow AI is a full-stack web application designed to simulate a smart hiring system. It allows users to input job descriptions and candidate details, then uses AI to evaluate and generate structured interview reports.

This project demonstrates the integration of modern web technologies with AI to solve real-world hiring challenges.

---

## ✨ Features

* 📄 Resume Analysis
* 🧠 AI-based Candidate Evaluation
* 📊 Interview Report Generation
* 🔍 Job Description Matching
* 🌐 Full-stack Web Application
* ⚡ Real-time Processing

## ⚙️ Installation & Setup

### 1. Clone the Repository

```
git clone https://github.com/your-username/hireflow-ai.git
cd hireflow-ai
```

---

### 2. Setup Backend

```
cd backend
npm install
npm start
```

---

### 3. Setup Frontend

```
cd frontend
npm install
npm run dev
```

---

## 🌍 Deployment

* Frontend deployed on Vercel
* Backend deployed on Render

Make sure to configure environment variables:

```
VITE_API_URL=your_backend_url
```

---

## 🔐 Environment Variables

Create a `.env` file in the backend:

```
PORT=3000
MONGO_URI=your_mongodb_connection
```
